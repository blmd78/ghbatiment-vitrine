# GH Bâtiment — Site vitrine

Site vitrine de GH Bâtiment, entreprise générale du bâtiment tous corps de métier en Île-de-France.

## Stack

- **Next.js** (Pages Router)
- **TypeScript**
- **Tailwind CSS v4**
- **Swiper** (galerie photos)

## Démarrage

```bash
pnpm install
pnpm dev
```

Le site est accessible sur [http://localhost:3000](http://localhost:3000).

## Build

```bash
pnpm build
pnpm start
```
